My telegram bot, @haodabot, what is it and what does it do?

If you want to check out my telegram bot, please go to the link in the navagation bar, but please read HOW TO USE IT before proceeding.

PURPOSE To allow me to update my friends on Telegram while I am on vacation in China, despite it being blocked by China's "great firewall".

HOW IT ACCOMPLISHES THIS It accomplishes this by sending information to a websocket, which acts as a median that can communicate freely with Telegram servers.

HOW TO USE IT I made a very simple 'password' program, to keep some of my friends from abusing the program, but the password is very simple to figure out, if you know how to inspect its element, hint hint. Feel free to test out my program, but please do not message anyone other than myself, or yourself, if you have a telegram account and know your chat_id.

SOURCE CODE? Unfortunately, I do not have the source code available on my github account. You see most of it by inspecting element and using the url: haodafan.com/botsource.js

DEVELOPMENT The development process for this program took place in the week before I left for China. I originally hosted my website on Hostgator.com; however due to its lack of freedom, vps service, and node.js service, I needed to switch it to a different hosting service, digitalocean, which is much more developer-oriented and gave me much more freedom. The telgram bot api for Node.js was heavily used in the making of the bot.
